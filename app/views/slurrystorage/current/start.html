Use this service to:

  • check if you can apply for a grant for your project (takes about 5 minutes)
  • estimate how much grant funding you could get (takes about 10 minutes if you have all the project details)

Who can apply

You can apply if you:

  • are a dairy, beef or pig farmer with an existing slurry system
  • your slurry store is in England
  • estimate the project costs are over £50,000
If your project is eligible, you can submit your answers to the Rural Payments Agency (RPA) to request the full application form.

Start now

Before you start

To  use the checker,  you need:

  • information about your business (for example, number of employees and turnover)
  • information about the project (for example, type of store and capacity, type of cover and size, approximate size and quantity of other items you need, the OS grid reference)
  • a list of the items you’d like to buy for the project

If you do not enter any information for more than 20 minutes, your application will time out and you  will have to start again. 

Problems using the online service
