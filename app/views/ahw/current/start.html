Use this service to:

  • check if you can apply for a grant for your project (takes about 20 minutes)
  • estimate how much grant funding you could get (takes about 20 minutes if you have all the project details)

You need to complete the checker in one session as you cannot save your information.

Who can apply

You can apply if you:

  • are a dairy, beef or cattle and arable farmer
  • will do the grant-funded work in England
  • are building new calf housing or extending or refurbishing existing calf housing

The minimum grant you can claim is £20,000 (40% of £50,000). The maximum grant is £500,000.

If your project is eligible, you can submit your answers to the Rural Payments Agency (RPA) to request the full application form.

Start now

Before you start

To  use the checker,  you need:

  • information about your business (for example, number of employees and turnover)
  • information about the project (for example, building structure, average floor space per calf, floor and bedding type)
  • an estimate of the total cost of the project

If you do not enter any information for more than 20 minutes, your application will time out and you  will have to start again. 

Problems using the online service

If you have any problems using the online  service, call the RPA and follow the options for the Farming Transformation Fund scheme.
